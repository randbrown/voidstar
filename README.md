# voidstar

A collection of music audio and visual coding experiments and projects.

## Project Structure

This repository contains various folders and files for audio and visual coding work:

- **audio/** - Audio coding experiments, synthesizers, and audio processing
- **visual/** - Visual coding, graphics, and visualization projects
- **experiments/** - Experimental and exploratory code
- **resources/** - Shared resources, samples, and utilities

## About

voidstar is an ongoing creative coding project exploring the intersection of audio and visual programming.
